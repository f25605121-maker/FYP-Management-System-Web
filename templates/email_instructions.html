How to Find Your Password Reset Link

Due to email sending restrictions, password reset links are saved to files instead of being sent via email in development mode.

1 Where to Find the Files

The reset link has been saved to a file in the emails folder in your project directory. Look for a JSON file with a name like:

email_YYYYMMDD_HHMMSS_youremail_example_com.json

The file name includes the date and time the reset was requested, along with your email address (with @ and . replaced with _).

2 Opening the File

You can open this JSON file with any text editor. The file contains information about the reset email, including:

{
  "to": "your.email@example.com",
  "from": "system@example.com",
  "subject": "Password Reset Request",
  "body": "To reset your password, visit the following link: http://...",
  "timestamp": "2023-05-24 09:45:23.456789"
}

3 Using the Reset Link

Copy the reset link from either:

  • The link shown directly on the forgot password page
  • The "body" field in the JSON file

Paste this link in your browser to go to the password reset page.
